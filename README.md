# BankAccount Challenge 

## Description
This project is a simple Java program that simulates a basic bank account.  
It demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, and methods.

## Features
- Deposit money into a bank account
- Validate deposit amount
- Check current account balance

## Technologies
- Java

## How the Program Works
1. A `BankAccount` class is created with a private `balance` variable.
2. The `deposit()` method adds money to the account if the amount is valid.
3. The `getBalance()` method returns the current balance.
4. The `main` method creates an account and performs a deposit.

## Example Output
```
Deposited: $150.9
Current Balance: $150.9
```

## Concepts Used
- Classes and Objects
- Encapsulation
- Methods
- Conditional Statements
